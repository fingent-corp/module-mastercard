The MasterCard module implements the integration with the MasterCard Payment Gateway Services and makes it available as a payment method in Magento.

## Feature Support

Magento 2 Mastercard Payment Gateway Service module supports following list of features:

- Card payments
- Alternative payment methods
- Hosted Session
- Hosted Checkout
- Full refunds
- Partial refunds
- AVS
- 3DSv1
- 3DSv2
- ACH
- Tokenisation (Only for Hosted Payment Session method)
- Certificate based authentication
